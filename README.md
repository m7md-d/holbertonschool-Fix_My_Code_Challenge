# Fix My Code Challenge

A set of small programs that each ship with one planted bug. The task is to find
the defect and fix it without rewriting the program. Each task is in a different
language: Python, JavaScript (Node), Ruby, and C.

## Layout

| Path | Contents |
|------|----------|
| [0x00-Fix_My_Code_Challenge-main/](0x00-Fix_My_Code_Challenge-main/) | The original sources, with the planted bugs |
| [challenge/](challenge/) | The fixed solutions, one per task |

See [challenge/README.md](challenge/README.md) for the per-task breakdown of each
bug and its fix, plus how to run them.
